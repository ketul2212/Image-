
# Text detection in images

- 1st run bellow command to Install Tesseract OCR 5 on Ubuntu 20.04|18.04:

```bash
  1 ) sudo apt update

  2 ) sudo add-apt-repository ppa:alex-p/tesseract-ocr-devel

  3 ) sudo apt install -y tesseract-ocr

  4 ) sudo apt update

  5 ) tesseract --version
```
